REQUISITOS PARA USO El usuario debe tener un dispositivo movil inteligente (smartphone), con sistema operativo Ios o Android para poder instalar y utilizar esta aplicación. El usuario se compromete a leer las condiciones y restricciones aca establecidas. El usuario acepta ingresar sus datos de manera voluntaria. El usuario que se registre debe ser mayor de edad El cliente autoriza al bar a acceder a sus datos y enviar a sus correos y celulares información relacionada con eventos del bar. Para acceder a la app el cliente debe tener un usuario y contraseña. OBLIGACIONES DEL USUARIO El cliente no debe utilizar la información compratida en la app para fines lucrativas o efectos contrarios a la ley, la moral o el orden público. eL cliente no debera reproducir, copiar o transformar la información o contenidos de la aplicación. El cliente no debe permitir que terceros manipulen la palicación. La presente aplicación etara disponible a nivel nacional. FUNCIONALIDAD Y CUSO DE LA APLICACIÓN La descarga de la presente aplicación es gratuita. BabylonApp esta destinada para informarle alos clientes los eventos que se van a realizar en el bar, para que hagan sus reservas, nos comentes sus quejas y/o sugerencias y para tener sus datos en nuestra base de datos.
